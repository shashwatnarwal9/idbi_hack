# Security notes

## Scope

आय·AI is a local hackathon prototype. It processes only synthetic data
generated by `aayai.datagen`; no real customer data is included or expected in
this repository.

## What is deliberately not in git

- `data/` (raw/bronze/silver/gold) and any `*.csv` / `*.parquet`: regenerate
  with `python -m aayai.datagen` followed by the pipeline stages.
- `model/` artifacts and reports: regenerate with `python -m aayai.model.train`.
- `.env` and any key/credential material: copy `.env.example` to `.env`.
- Terraform state (`*.tfstate*`, `.terraform/`) is ignored pre-emptively;
  state files can embed secrets and must never be pushed.

## Known local-dev credentials

`docker-compose.yaml` contains hardcoded development credentials
(`airflow`/`airflow` for the Airflow metadata database, `aayai`/`aayai` for the
serving store, `admin`/`admin` for the Airflow UI). They bind to localhost and
exist only for local demos. Rotate to managed secrets (environment variables
or a secrets manager) before any shared or cloud deployment.

## Reporting

This is a demo repository; open an issue for anything that looks like a
leaked secret or real data.
